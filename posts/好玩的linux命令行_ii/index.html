好玩的Linux命令行_II

2015, Jun 09

这条网站上统计了很多很多好玩有用的命令，感兴趣的同学可以看一下哦！ http://www.commandlinefu.com/commands/browse/sort-by-votes/

下面这些是我从上面摘抄的，但是也请转载注明出处～

先装上pv，然后再敲这一条，会把你帅呆了的！

    echo "You can simulate on-screen typing just like in the movies" | pv -qL 10

来，测试一下咱的硬盘有多快：

timeout 2s yes "hello world!" > big_file & watch du -h big_file

常用的20条命令++

history | awk '{a[$2]++}END{for(i in a){print a[i] " " i}}' | sort -rn | head

文字版星球大战!!!

telnet towel.blinkenlights.nl

在终端右上角显示个表

去tmp目录下逛一圈再回来(小括号代表子shell)

(cd /tmp && ls)

nc用来传文件：

nc -v -l 80 < file.ext

更改文件后缀名，把wrfly.png改成wrfly.jpg：

mv wrfly.{png,jpg}

把命令的输出内容转换成图片，超酷炫：

ifconfig | convert label:@- ip.png

vim加密：

vim -x <FILENAME>

(如果需要加密已经存在的文件，则需要保存成另一个文件，:wq + new_name，然后下次打开 new_name 的时候就需要输入密码了)

看一下你家的目录都是干啥用的：

man hier

来看看都是那些程序在联网：

ss -p

用openssl给文件加密：

openssl [methods] [-salt] -in file -out enc_file

openssl aes-256-ecb -salt -in file -out enc_file

解密

openssl aes-256-ecb -d -in enc_fiel -out dec_file

在来个好玩的，二进制的表，简称，二表

提醒自己在十五分钟之后滚开：

leave +15
(sudo apt-get install leave)

(玩过之后就觉得不是那么好玩了- -)

把Ctrl+l和echo hi绑定到一起，你按下C+l的时候就会echo hi～

bind -x '"\C-l":echo hi'

如果想要永久绑定，可以把command写入~/.bashrc

‘most’，一个有颜色的‘man’

目前就这么多啦，前十页我挨个尝试的，希望能给你带来快乐～
