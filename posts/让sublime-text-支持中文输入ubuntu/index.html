让sublime text 支持中文输入（ubuntu）

Posted on 2015, Jun 05

解决在ubuntu下用 sublime_text_3 输入中文的问题。

只解决问题，不废话：

cd /opt/sublime_text &&\
sudo  wget "https://gist.githubusercontent.com/wrfly/baf1314ae25f41fe8cde/raw/45c669808bed8eb6d98fe6caa3f730076e5aede3/imfix.c" &&\
sudo apt-get install build-essential libgtk2.0-dev &&\
sudo gcc -shared -o imfix.so imfix.c  `pkg-config --libs --cflags gtk+-2.0` -fPIC &&\
sudo sed -i "s/exec/LD_PRELOAD=\/opt\/sublime_text\/imfix.so/" /usr/bin/subl &&\
sudo sed -i "s/\/opt\/sublime_text\/sublime_text/subl/g" /usr/share/applications/sublime_text.desktop &&\
echo "Succeed!"

如果你还嫌麻烦的话，没关系～

一键安装脚本在这儿：

wget https://gist.githubusercontent.com/wrfly/5f4187be96d93e95ad07/raw/15953c30b549efb4185df6d229324e465d279606/auto_imfix.sh

然后运行就好了 :)
